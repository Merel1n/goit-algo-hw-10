# Обчислення визначеного інтеграла методом Монте-Карло

У даній роботі було реалізовано алгоритм обчислення визначеного інтеграла
функції f(x) = x² на відрізку [0, 2] за допомогою методу Монте-Карло.

Метод Монте-Карло полягає у випадковій генерації точок у прямокутнику,
що покриває область під графіком функції, та підрахунку частки точок,
які потрапляють під криву.

Отримане значення інтеграла було порівняно з результатом, отриманим:
- аналітичним способом
- за допомогою функції quad з бібліотеки SciPy

Аналітичне значення інтеграла дорівнює 8/3 ≈ 2.6667.
Результат, отриманий методом Монте-Карло, близький до аналітичного та
збігається з результатом функції quad з незначною похибкою.

Зі збільшенням кількості випадкових точок точність методу Монте-Карло
зростає, що підтверджує правильність реалізації алгоритму.
